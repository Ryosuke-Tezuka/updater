# アプリケーションの概要

学習の実績がグラフで視覚化されるオンライン学習サイト `"Updater"`

# アプリケーションの機能一覧

* ログイン機能（ユーザー/管理者）
* 動画投稿、再生機能
* 学習実績のグラフ化機能
* ページネーション機能
* 検索機能
* レスポンシブ対応

# アプリケーション内で使用している技術一覧

* OS - Linux(Ubuntu)
* 言語 - HTML,CSS,JavaScript,Ruby,SQL
* フレームワーク - Ruby on Rails, JQuery
* インフラ - AWS(EC2,RDS,EIP)
* その他ツール - chart.js, carrier wave, bootstrap
